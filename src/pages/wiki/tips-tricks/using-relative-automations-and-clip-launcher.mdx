# Using relative automations and clip launcher

Using relative automations (additive/multiplicative) you can contain any adjustments you make within that clip alone. This will prevent automations from sticking. This is great for very local sound design stuff and ensures your clips will sound the same every time, that is unless you tweak the base parameter values of course.
