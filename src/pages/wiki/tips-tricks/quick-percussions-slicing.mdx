# Quick percussions/slicing

Add a random sample to sampler, set ADSR to quick stabs. Now add a 12-pitch mod, map each note to the play/position offset and try to find some cool sounds. Done, you can now keep mapping parameters to each note of course, add effects, change the ADSR, go nuts.
